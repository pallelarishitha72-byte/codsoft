# codsoft
CODSOFT strongly believes in the power of leadership to drive positive change. Through mentorship programs, workshops, and collaborative projects, it empowers individuals to develop leadership skills and make meaningful contributions in their respective fields.

 About This Repository

This repository contains all the Data Science internship tasks completed under CODSOFT.
It is created as part of the internship requirement and showcases hands-on experience in:
Data Analysis
Machine Learning
Model Building
Real-world problem solving
All tasks are implemented using Python and popular data science libraries.

 Internship Objectives
Gain practical experience in Data Science
Apply machine learning algorithms to real datasets
Improve skills in data preprocessing & visualization
Build industry-level projects
Create a strong GitHub & LinkedIn portfolio

 Tasks Completed
 Task 1: Titanic Survival Prediction
Built a machine learning model to predict passenger survival
Used features like age, gender, class, fare, etc.
Applied data preprocessing, EDA, and classification algorithms
Dataset:
https://www.kaggle.com/datasets/yasserh/titanic-dataset

 Task 2: sales Prediction with Python
Forecasted product sales using advertising data
Applied regression techniques
Task 3: Iris Flower Classification
Classified iris flowers into Setosa, Versicolor, and Virginica
Used sepal and petal measurements
Implemented supervised classification models
Datasets:
https://www.kaggle.com/datasets/arshid/iris-flower-dataset
